# JavaScript Training Projects

This repository contains a collection of beginner-friendly JavaScript projects for learning and practice. Each folder is a self-contained project with its own code and resources.

## Projects Overview

### 01-ToDoList
A simple ToDo List app using HTML, CSS, JavaScript, and [JSON Server](https://github.com/typicode/json-server) for a mock backend.

### 02-Blog
A basic blog project with multiple scripts and a mock database. Includes API examples and Promise utilities.

### 03-Form validation
A form validation demo showing how to validate user input using JavaScript.

## How to Use
1. Browse each project folder for its own README or instructions.
2. For projects using JSON Server, install it globally:
   ```bash
   npm install -g json-server
   ```
3. Follow the instructions in each project to run and test locally.

**Note:** These projects are for educational purposes and demonstrate core JavaScript concepts.
