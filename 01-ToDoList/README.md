# ToDo List App

This is a simple ToDo List application built with HTML, CSS, and JavaScript.

## Features
- Add, delete, and manage your tasks
- Data persistence using a local JSON file
- Uses [JSON Server](https://github.com/typicode/json-server) to simulate a REST API backend

## Getting Started
1. Install JSON Server globally (if not already installed):
   ```bash
   npm install -g json-server
   ```
2. Start the JSON Server:
   ```bash
   json-server --watch db.json
   ```
3. Open `index.html` in your browser to use the app.

## Folder Structure
- `app.js` - Main JavaScript logic
- `db.json` - Mock database for JSON Server
- `index.html` - Main HTML file
- `styles.css` - Stylesheet
