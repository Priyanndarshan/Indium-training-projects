# Form Validation Project

This folder contains a simple form validation application using HTML, CSS, and JavaScript. The project demonstrates client-side validation techniques for web forms.

## Structure

- `index.html`: Main HTML file containing the form.
- `script.js`: JavaScript file implementing form validation logic.
- `styles.css`: Stylesheet for form and validation UI.

## Features

- Real-time validation for form fields
- User-friendly error messages
- Customizable validation rules
- Responsive design

## Usage

1. Open `index.html` in your browser to view and test the form.
2. Modify `script.js` to add or change validation rules as needed.

## Requirements

- Modern web browser
- No external dependencies required
